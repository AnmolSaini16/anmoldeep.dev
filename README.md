## [anmoldeep.dev](https://anmoldeep.dev)

- Next.Js
- Framer Motion
- Tailwind CSS
- Resend
- MDX for blogs
- Vercel Deployment

## Getting Started

Install dependencies

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
